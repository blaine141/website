EM Order Page

August 2014 - September 2015

    The Electrimotion order system is a basic order system which allows the customer to order a product and supply any other notes about the product.

    All the orders and products are stored in a MySQL database and accessed through a PHP page. The MySQL Database can be used to change the website without even touching any HTML or PHP code.

    Another cool feature that I made is the UPS Tracking link. Using POST, this sends the tracking number to the UPS page it opens to show you the tracking information.
